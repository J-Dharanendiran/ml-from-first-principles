# 🧠 Machine Learning from First Principles

> *"Machine Learning is not about teaching computers the rules. It's about letting computers learn the rules from data."*

Welcome to **Machine Learning from First Principles**.

This repository documents my journey of learning Machine Learning by understanding **why things work**, not just **how to use them**.

Instead of memorizing algorithms or blindly importing libraries, every concept is explored through **intuition**, **mathematical foundations**, **Python implementations**, **visual explanations**, and **real-world applications**.

I believe that true learning begins with **first principles**. Every topic in this repository is broken down to its fundamentals before building toward more advanced concepts.
